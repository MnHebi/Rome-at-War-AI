# Rome-at-War-AI
Mod to introduce AI that knows how to use new units and mechanics to the Age of Empires 2: Definitive Edition mod "Romae Ad Bellum" or "Rome at War".</br>
Built upon the AIBuilder AI in AoE2: DE Files.
